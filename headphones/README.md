# Smiling School - Headphones Project  

## Project Overview  
This project implements a responsive web page based on a provided Figma design. It focuses on creating an engaging, user-friendly experience with proper layout, typography, and interactivity.

## Getting Started  

To access the design details:  
1. **Figma Access**:  
   - Create a Figma account if you don’t have one.  
   - Open the [Figma Project](https://www.figma.com/design/ZJvo7TvmYXP2ML7iwuoUov/a5366bbd595c643993665e2a28909370a7e12c66-(Copy)?t=YQ3C2cKWQnrFDvIw-0) 
   - Click **Duplicate to your Drafts** to access all design elements.

2. **Fonts**:  
   - Download and install the following fonts to match the design:  
     - [Source Sans Pro](https://fonts.google.com/specimen/Source+Sans+Pro)  
     - [Spin Cycle OT](https://fonts.adobe.com/fonts/spin-cycle)  

## Features  

### Responsive Design  
- The web page adapts seamlessly to different screen sizes.  
- Switches to the mobile version when the screen width is `480px` or less.  

### Interactions  
- **Link hover/active**: Color changes to `#FF6565`.  
- **Button hover/active**: Opacity changes to `0.9`.  

### Layout  
- Maximum content width: `1000px`, centered on the page for a clean and balanced look.
